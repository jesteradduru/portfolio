Smart Brain

Smart Brain app is an Face Recognition app. In order to use its functionalities, one must register first. After registration you can find any image url and paste it in the form to detect any face. It uses Clarifai API to detect a face in a picture.

This app was created with React.js and designed with Bootstrap 4 for its front-end and for its back-end, I used Node.js for the server and Postgres for the database. The server is hosted in Heroku and the front-end is hosted on github.
